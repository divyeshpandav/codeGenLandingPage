NoKode is a cross-platform source code generation engine.

NoKode lets you build your own code generation templates with TypeScript. It consists of a Node.js CLI and extensible APIs, making it easy for developers to create, share and re-use code generators for their favorite programming languages and frameworks. And it's entirely free and open source.
